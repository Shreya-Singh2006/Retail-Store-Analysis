# Retail Store Data Analysis

## Overview
This project analyzes retail store data to uncover sales trends, customer behavior, product bundling patterns, and profitability. The insights are presented through an interactive **Power BI dashboard**, designed to help store managers make data-driven decisions.

## Objectives
- Clean and preprocess raw retail datasets (Python + Power Query).
- Perform exploratory analysis: sales trends, top products, customer segmentation.
- Apply advanced techniques: basket analysis, profitability evaluation.
- Visualize KPIs in Power BI dashboards.
- Provide a framework for future extension into a web application (Streamlit).

## Workflow
1. **Data Collection** – Kaggle retail datasets or store-provided files.
2. **Data Cleaning** – Python scripts handle missing values, duplicates, and transformations.
3. **Analysis Modules**  
   - Sales Trends  
   - Top Products  
   - Customer Segmentation  
   - Basket Analysis  
   - Profitability Analysis  
4. **Visualization** – Power BI dashboard with interactive charts and KPIs.
5. **Future Scope** – Streamlit app where managers upload raw data and see results instantly.

## Features
- **Sales Trend Line Chart** – Revenue over time.
- **Top Products Bar Chart** – Best-selling items.
- **Customer Segmentation Pie Chart** – Contribution by demographics.
- **Profitability Summary** – Revenue vs. cost margins.
- **Basket Analysis** – Product bundling recommendations.
